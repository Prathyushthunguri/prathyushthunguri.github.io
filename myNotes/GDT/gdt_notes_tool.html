Limits & Fits — Concepts
ISO / IS (Indian Standard) System — Fundamental definitions from IIT Bombay ME 423 notes
Zero Line & Basic Size
Zero Line: Represents the basic size — the reference line from which upper and lower deviations are measured. Above = positive deviation; below = negative deviation.

Basic Size (D): The theoretically exact size used as the origin for limits of size. Both hole and shaft limits are defined relative to this value.
Deviations — ES, EI, es, ei
ES (Upper Deviation, Hole): Algebraic difference between max hole size and basic size.
EI (Lower Deviation, Hole): Algebraic difference between min hole size and basic size.
es (Upper Deviation, Shaft): Algebraic difference between max shaft size and basic size.
ei (Lower Deviation, Shaft): Algebraic difference between min shaft size and basic size.

Tolerance = ES − EI (hole)  |  es − ei (shaft)
Fundamental Deviation (FD)
One of the two deviations chosen to define the position of the tolerance zone relative to the zero line. It is either the upper or lower deviation — whichever is closer to the zero line.

Holes (uppercase): A–G (EI +ve above zero), H (EI=0), JS (bilateral), J–ZC (ES −ve below zero).
Shafts (lowercase): a–g (es −ve below zero), h (es=0), js (bilateral), k–zc (ei +ve above zero).
Types of Fit
Clearance Fit: Hole always larger than shaft. Always positive clearance (gap).
Max Clearance = Dmax,hole − dmin,shaft
Min Clearance = Dmin,hole − dmax,shaft

Transition Fit: May result in either small clearance or small interference.

Interference Fit: Shaft always larger than hole. Always negative clearance (force/press fit).
Max Interference = dmax,shaft − Dmin,hole
Systems of Fit
Hole Basis System (preferred): The hole is kept constant (H — zero lower deviation). Different shafts are used to achieve the required fit. Economical — one standard reamer for hole; shaft adjusted.

Shaft Basis System: The shaft is kept constant (h — zero upper deviation). Different holes are used. Used when a single shaft runs through multiple features of different fits.
Accuracy vs. Precision
Precision: Repeatability of measurements — how closely repeated measurements agree with each other.

Accuracy: How close a result is to the true value.

Accuracy² = Systematic Error² + Repeatability²
where: Systematic error = True value − Mean of readings
Limits of Size — Calculation Steps
STEP 1
Select fit type based on functional requirement (clearance / transition / interference)
STEP 2
Select hole and shaft designations (e.g. H7 hole, g6 shaft for clearance)
STEP 3
Find diameter step for basic size D. Compute D = √(a×b) geometric mean of step limits
STEP 4
Compute standard tolerance unit: i = 0.45·D^(1/3) + 0.001·D (µm, D in mm)
STEP 5
Compute IT grade tolerance: e.g. IT6 = 10i, IT7 = 16i, IT8 = 25i (round to nearest µm)
STEP 6
Compute fundamental deviation using formula (e.g. shaft g: es = −2.5D^0.34 µm). Round per rounding rules.
STEP 7
Evaluate limits:
Hole: D_max = Basic + ES; D_min = Basic + EI
Shaft: d_max = Basic + es; d_min = Basic + ei
STEP 8
Determine fit type and compute Max/Min clearance or Max/Min interference
IT Grades of Tolerance
ISO 286 / IS standard — 18 grades from IT01 to IT16. Tolerance unit i = 0.45·D^(1/3) + 0.001·D (µm)
Standard Tolerance Unit Formula
i (µm) = 0.45 × ∛D + 0.001 × D where D (mm) = geometric mean of lower and upper limits of the diameter step = √(a × b)

Diameter Steps (ISI): 1–3, 3–6, 6–10, 10–18, 18–30, 30–50, 50–80, 80–120, 120–180, 180–250, 250–315, 315–400, 400–500 mm
